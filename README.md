# Placement Portal

## Overview

Placement Portal is a MERN Stack web application developed to simplify and manage campus placement activities. The system provides a platform for students to apply for jobs, upload resumes, and track applications, while administrators can manage companies, announcements, and placement records.

## Features

### Student Features

* Student Registration and Login
* Profile Management
* Resume Upload
* View Available Companies
* Search and Filter Companies
* Apply for Jobs
* Track Application Status
* View Announcements

### Admin Features

* Admin Authentication
* Add, Edit, and Delete Companies
* Open/Close Company Applications
* View Company Applicants
* Manage Applications
* Post Announcements
* Placement Statistics Dashboard

## Technology Stack

### Frontend

* React.js
* React Router DOM
* Axios
* Bootstrap
* React Toastify

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Deployment

* Frontend deployed on Vercel
* Backend deployed on Render
